## 📁 Task Summaries

### 📂 Task 1 — Term Deposit Subscription Prediction
**Objective:** Predict if a bank customer will subscribe to a term deposit.
- Models: Logistic Regression + Random Forest
- Evaluation: Confusion Matrix, F1-Score, ROC Curve
- Advanced: SHAP used to explain 5 individual predictions
- 📄 [View Task](./Task1-Term-Deposit-Prediction/)

---

### 📂 Task 2 — Customer Segmentation
**Objective:** Group mall customers by spending behavior.
- Algorithm: K-Means Clustering (K=5 via Elbow Method)
- Visualization: PCA for 2D cluster plot
- Output: Marketing strategies per customer segment
- 📄 [View Task](./Task2-Customer-Segmentation/)

---

### 📂 Task 3 — Energy Consumption Forecasting
**Objective:** Forecast household electricity usage.
- Models compared: ARIMA vs Prophet vs XGBoost
- Features: Lag values, rolling averages, time-based features
- Evaluation: MAE and RMSE comparison
- 📄 [View Task](./Task3-Energy-Forecasting/)

---

### 📂 Task 4 — Loan Default Risk
**Objective:** Predict loan defaults and minimize business losses.
- Model: CatBoost Classifier
- Business Logic: Cost matrix (FN=$10,000 vs FP=$500)
- Key Result: Optimal decision threshold found via cost optimization
- 📄 [View Task](./Task4-Loan-Default-Risk/)

---

### 📂 Task 5 — Interactive Sales Dashboard
**Objective:** Build an interactive BI dashboard for sales analysis.
- Tool: Streamlit web app
- Features: Region/Category/Sub-Category filters
- KPIs: Total Sales, Profit, Top 5 Customers
- 📄 [View Task](./Task5-Sales-Dashboard/)
